# ABSTRACT

Eduguide is a full-stack, AI-driven study planner and educational dashboard platform developed using React (frontend) and Node.js/Express (backend) with MongoDB as the database. The platform provides a centralized space where students and self-directed learners can upload syllabus documents (either in raw text format or as image files), generate structured study topics, track study progress, and interact with an integrated conversational AI tutor.

The system employs a token-based authentication model using JSON Web Tokens (JWT) for secure session management. Registered users can create personalized study folders, import and analyze syllabi, start focused study sessions, and review key concepts. When a user uploads a syllabus document, the integrated AI automatically parses it to extract individual topics, estimating a study duration in minutes and providing concise, high-yield study notes.

The interactive dashboard provides comprehensive tracking and management capabilities. It features real-time study statistics and progress charts to monitor completed work, a built-in study timer for focused sessions, and a history of study folders. An AI chatbot assistant is integrated directly into the workspace, allowing users to ask academic questions, get clear explanations of topics, and receive personalized study advice. The workspace is designed to be engaging, featuring visual toast notifications and celebration animations when study topics are successfully completed.

The backend is built using Node.js and Express.js, with Mongoose for MongoDB interaction and database object modeling. The platform integrates the Google Generative AI SDK, utilizing the `gemini-2.5-flash` model for conversational chat and structured JSON syllabus analysis. RESTful APIs handle all client-server communication, including authentication (with bcryptjs password hashing), conversation history persistence, and syllabus management. The frontend is built with React and Vite, utilizing Tailwind CSS for styling and modern component libraries (including Radix UI and Material-UI) for responsive layout management.

This project demonstrates the practical application of modern full-stack web development, AI integration (multimodal parsing and structured output generation), RESTful API design, database modeling, secure authentication, and modern user experience (UX) design in solving a real-world educational problem.
